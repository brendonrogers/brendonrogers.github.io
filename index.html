Deploying static Apache pages via git

A journey through standing up a website using git.

Environment:

  • AWS Ubuntu Linux host
  • Apache2
  • git
  • Dev machine running MacOS Sierra
Preparing git

Prepare Apache

Give user called 'ubuntu' write permissions to the /var/www/html/ directory which is the web root and where git will write the code to

sudo adduser ubuntu www-data
Adding user 'ubuntu' to group 'www-data' ...
Adding user ubuntu to group www-data
Done.

sudo chown -R www-data:www-data /var/www/html/
sudo chmod -R g+rwX /var/www/html/
